react router dom 검색.

router를 사용하면 BrowserRouter, Switch, Route, Link 기능 사용 가능.

터미널
npm install react-router-dom@5.2.0

http://localhost:3000/ =>시작페이지 보여줘
http://localhost:3000/Detail/ => 상세페이지 시작화면 보여줘
http://localhost:3000/Detail/01 => 01제품의 상세페이지 시작화면 보여줘
