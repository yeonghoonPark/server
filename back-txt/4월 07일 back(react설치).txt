nodejs 검색 및 설치
→ 16.14.2 LTS 다운로드, 안된다면 최신버전.
→ 기본값으로 설치
→ cmd 또는 vscode의 터미널에서  'node -v'로 버전 확인가능

react 실행
→ vscode실행, 작업할 폴더 생성
→ vscode의 터미널 실행, npx create-react-app '폴더명' 또는 .
→ vscode의 터미널 실행, npm start

react bootstrap
→ vscode의 터미널 실행, npm install react-bootstrap bootstrap
→ App.js에서 import 
 import { Button, etc... } from 'react-bootstrap';
 import 'bootstrap/dist/css/bootstrap.min.css';
